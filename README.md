# Glass_box_chat
Glass Box AI Chat – a transparent alternative to traditional black-box AI systems. Visualize how agents think, call tools, and collaborate in real time through an interactive execution trace inspired by modern agent runtimes.

## Quick start

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements-dev.txt
python -m glass_box_chat
pytest
```
